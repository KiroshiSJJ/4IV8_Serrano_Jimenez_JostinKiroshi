Objetivo del Portafolio

Este portafolio recopila las evidencias de las prácticas desarrolladas durante el curso, demostrando la aplicación de los conocimientos obtenidos en desarrollo web.

Aprendizajes

  • Dominio de estructuras HTML.
  • Implementación de estilos CSS bajo guías de diseño (Figma).
  • Lógica de programación aplicada con JavaScript.
  • Desarrollo de validaciones de formulario.
  • Gestión de datos mediante operaciones CRUD.
